AIS Data for 2019

Below are links to the files that make up the AIS dataset for 2019. This HTML file is here to emulate the access you would have gotten through an ftp site. The files containing geospatial indices, metadata, etc., are listed first, followed by the files. To download in bulk, it is suggested you use a program such as wget (example below).

Example to download entire data set using wget (windows version at http://gnuwin32.sourceforge.net/packages/wget.htm):
wget -np -r -nH -L --cut-dirs=3 https://coast.noaa.gov/htdata/CMSP/AISDataHandler/2019/
This will give you the directory structure starting at 2019. Modify the --cut-dirs=3 argument to modify the start directory (e.g. --cut-dirs=2 will start at one directory earlier). Wget is a command-line program and the output will be to the directory you run it in. More options and information can be found at http://www.gnu.org/software/wget/manual/wget.html. The other options in the example mean:

  • -np = don't grab the parent directory
  • -r = recursive (descend into directories)
  • -nH = don't generate host-prefixed directories
  • -L = follow relative links only (so you don't download wget just because it's linked here)

If you have problems with wget that look like an authorization or certificate issue, you may need to add the option "--no-check-certificate".

The total size to download all the files is 99G.
